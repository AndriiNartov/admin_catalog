Hi! This is the main page of simple test application. The task is to create application, using Flask framework and two related libraries:

  • Flask-Admin
  • Flask-Security
Using flask-admin I had to implement the catalog of some products with those specifications:
  • color
  • weight
  • price
This catalog also must include address of delivery and some filters, which we can use for filtering addresses in catalog by:
  • country
  • city
  • street
Authorization to admin-catalog must have been done using flask-security. For convenience, by default, in database is already created user. The credentials are:
  • Email: admin@example.com
  • Password: admin
You can log in using these credentials or create your own user here.
